Новые заявки

Каждую новую заявку лаборатория сначала должна принять в работу

Прокрутите таблицу вправо, чтобы увидеть столбик таблицы "Действия", в этом столбике есть желтая кнопка , нажмите ее, чтобы переместить заявку в работу, статус заявки сменится и вы увидите сообщение об этом, заявка будет продолжать отображаться в этой таблице, пока вы не уйдете со странички или не выполните поиск по таблице.

Когда заявка находится в работе, ее статус можно сменить на статус "Готов", чтобы переместить заявку в "Готовые стыки"

Заявки в работе

Прокрутите таблицу вправо, чтобы увидеть столбик таблицы "Действия", в этом столбике есть зеленая кнопка кнопка , нажмите ее, чтобы переместить заявку в готовые, статус заявки сменится и вы увидите сообщение об этом, заявка будет продолжать отображаться в этой таблице, пока вы не уйдете со странички или не выполните поиск по таблице.




Заключения на заявку на стык

В левом меню открыть пункт меню Заключения (акты)

Заключения (акты)

Вы увидите таблицу заключений. В правом верхнем углу нажать на кнопку

Создание заключения

Вы увидите форму создания заключения на заявку на стык. Форма содержит в себе 4 подраздела на каждое из заключений по заявке: Акт ВИК, Заключение РК, Заключение ПВК, Заключение УЗК

После сохранения заключения вы увидите сверху кнопки для каждого акта (заключения) для открытия PDF файлов, также вы увидите кнопку перехода к редактированию заявки на стык по этому заключению

Поле "Номер заявки на стык" заполняется из выпадающего списка с помощью поиска. Если заявки в списке нет, тогда необходимо перейти в раздел помощи

Сварщик (исполнитель) - Создание заявки на стык

Поле "Дата акта (заключения)" заполняется вручную

Акт ВИК

Галочка "Проведен ВИК" выбирается, если была проверка ВИК

Поле "Дефекты" заполняется вручную

Поля "Контроль выполнил", "Руководитель" заполняется из выпадающего списка дефектоскопистов с помощью поиска. Если дефектоскописта в списке нет, тогда необходимо перейти в раздел помощи

Создание дефектоскописта

Заключение РК

Галочка "Проведен РК" выбирается, если была проверка РК

Поле "Дефекты" заполняется вручную

Необходимо нажать кнопку "Добавить снимок" нужное количество раз (согласно сделанным снимкам), затем прямо в таблице нажать на каждое поле и заполнить его. Поля для каждого снимка в таблице заполняются вручную - просто нажмите на нужное поле и начинайте вводить информацию. В столбце "Действия" есть красный крестик для каждого снимка - это удаление строки

Поля "Стык принял по внешнему виду", "Контроль произвел", "Заключение выдал" заполняются вручную из выпадающего списка дефектоскопистов с помощью поиска. Если дефектоскописта в списке нет, тогда необходимо перейти в раздел помощи

Создание дефектоскописта

Заключение ПВК

Галочка "Проведен ПВК" выбирается, если была проверка ПВК

Поля "Вид контроля", "Оценка качества" заполняются вручную из выпадающего списка. В случае, отсутствия необходимых элементов в выпадающем списке, необходимо обратиться к администратору, самостоятельно вы их добавить не сможете

Поле "Дефекты" заполняется вручную

Поля "Начальник", "Дефектоскопист" заполняются вручную из выпадающего списка дефектоскопистов с помощью поиска. Если дефектоскописта в списке нет, тогда необходимо перейти в раздел помощи

Создание дефектоскописта

Заключение УЗК

Галочка "Проведен УЗК" выбирается, если была проверка УЗК

Поля "Угол ввода луча в градусах", "Условная чувствительность", "Дефекты", "Примечание" заполняются вручную

Поле "Оценка качества" заполняется вручную из выпадающего списка. В случае, отсутствия необходимых элементов в выпадающем списке, необходимо обратиться к администратору, самостоятельно вы их добавить не сможете

Поле "Оператор", "Стык принял по внешнему виду", "Контоль произвел", "Заключение выдал" заполняются вручную из выпадающего списка дефектоскопистов с помощью поиска. Если дефектоскописта в списке нет, тогда необходимо перейти в раздел помощи
