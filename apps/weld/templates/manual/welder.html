Новые заявки

Вы увидите таблицу новых заявок на стыки. В правом верхнем углу нажать на кнопку

Создание заявки на стык

Вы увидите форму создания заявки на стык. Пожалуйста, заполняйте форму внимательно, редактировать заявку будет возможно только при наличии прав, по умолчанию, исполнители (сварщики) не имеют прав на редактирование

Поля "Сварное соединение", "Свариваемые элементы", "Материал", "Вид сварного соединения", "Тип сварки", "Вид контроля", "Категория трубопровода" заполняются из выпадающего списка. В случае, отсутствия необходимых элементов в выпадающем списке, необходимо обратиться к администратору, самостоятельно вы их добавить не сможете

Поля "Дата заявки на НК", "Заявка №" заполняются автоматически, редактировать их вы не можете

Вы можете загружать файлы/изображения через нажав на ссылку "выберите файл для загрузки", либо просто перетащить файл в серое поля с этой надписью. Загружать файлы можно будет и после сохранения заявки.

Поле "Заявку принял" на данном этапе заполнять не нужно, это поле будет заполнено, когда лаборатория примет заявку в работу

Чтобы сохранить заявку, необходимо нажать зеленую кнопку "Сохранить", если ошибок не будет, то заявка сохранится, в противном случае вы увидите красные предупреждения над полями, которые необходимо заполнить

Если заявка успешно сохранена в правом верхнем углу вы увидите, что теперь заявку можно посмотреь в PDF - синяя кнопка PDF заявка на стык

Поле "Сварное соединение" вы должны выбрать с помощью поиска из списка, чтобы найти нужный стык, вы можете ввести номер титула, номер линии, номер стыка, тогда вы сможете выбрать из найденных вариантов нужный стык. Если стык еще не создан, тогда необходимо перейти в раздел помощи

Создание стыка

Поле "Толщина стенки, мм" заполняется вручную

Поле "Диаметр трубопровода, мм" заполняется вручную

Поля "Сварщик 1", "Сварщик 2", "Сварщик 1 по факту", "Сварщик 2 по факту" заполняются с помощью поиска сварщика из выпадающего списка, наберите, фамилию и/или клеймо и выберите нужного сварщика. Если сварщика в списке нет, тогда необходимо перейти в раздел помощи

Создание сварщика

"Сварщик 1 по факту", "Сварщик 2 по факту" нужны, для внутреннего пользования, в случае, если работы производили сварщики, которые не могут быть указаны в заявке, в других случаях эти поля заполнять не нужно

Поле "Дата сварки" заполняется вручную, выбирается дата, когда была выполнена работа

Поле "Заявку подал" заполняется из выпадающего списка с помощью поиска, по умолчанию будет выставлен пользователь, который создает заявку (Вы), вы должны выбрать пользователя, который создан в системе. В случае, если пользователя нет в системе, необходимо перейти в раздел помощи

Создание пользователя

Поле "Примечание" заполняется вручную
